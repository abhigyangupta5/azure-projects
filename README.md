# Azure Translator Web App

A modern translator web application built using HTML, CSS, JavaScript and Microsoft Azure Translator API.

The application allows users to:

- Translate text between multiple languages
- Swap source and target languages
- Copy translated text
- Paste text from clipboard
- Use speech-to-text input
- Listen to translated output
- View translation history
- Receive popup notifications

---

# Features

- Modern glassmorphism UI
- Responsive design for desktop and mobile
- Microsoft Azure Translator integration
- Translation history stored in localStorage
- Voice recognition using Web Speech API
- Text-to-speech output
- Animated popup notifications
